# ScaleAssembly Homebrew Tap

## ZShot

```sh
brew install --cask scaleassembly/tap/zshot
```

Installs [ZShot](https://zshot-app.com) into `/Applications` and puts the
`zshot` CLI on your `PATH`. Requires an Apple Silicon Mac running macOS 26+.

The cask in this repo is updated automatically by the ZShot release process.
